参见

如果你刚开始接触 Python, 您可能希望首先了解该语言是什么。 Django是100％的Python， 所以如果你对Python的了解不多，你可能会从Django中获得更多的东西。

如果您刚开始接触编程，您可能需要从 非编程人员的Python资源列表 开始

如果您已经掌握其他几种语言，并希望快速开始使用Python， 我们建议您看看 Dive Into Python. 如果这不是你喜欢的风格，这里还有很多 books about Python.
